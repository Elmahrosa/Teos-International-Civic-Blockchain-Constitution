# Code of Conduct

This project follows a Contributor Covenant inspired Code of Conduct.
# Code of Conduct
- Petition-first, badge-gated participation.
- Respect governance votes and audit trails.
- Zero tolerance for fraud or malicious proposals.

# Contributing
- Sign petition; obtain badge.
- Use `governance/proposal_schema.json` for proposals.
- Log changes in `audits/transparency_log.md`.

## Our Pledge
We commit to providing a harassment-free experience for everyone.

## Expected Behavior
- Be respectful and constructive.
- Use inclusive language.
- Respect differing viewpoints and experiences.

## Unacceptable Behavior
- Harassment, derogatory comments, threats.
- Doxxing, leaking personal data, or harassment.

## Enforcement
Report incidents to maintainers (see repo contact). Violations may result in removal from project and reporting to platform moderators.
