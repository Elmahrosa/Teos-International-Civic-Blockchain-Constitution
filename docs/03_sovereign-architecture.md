## Architectural Principle

**Authority flows downward. Execution flows upward.**

- Constitution defines legitimacy  
- Governance defines permission  
- Compliance defines legality  
- Guard & Audit define safety and evidence  
- Kernel executes approved actions  
- APIs expose controlled access  

No layer may override a higher layer.
